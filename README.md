# django-hotel_booking-app
A full-stack Hotel Booking Web Application built using Django that allows users to search hotels, view details, and make bookings seamlessly. The system also includes an admin/dashboard interface to manage hotels and bookings.

🚀 Live Demo
https://django-hotel-booking-app.onrender.com

📌 Features
 -> User registration and login system
 -> Browse available hotels
 -> View detailed hotel information
 -> Book rooms online
 -> Admin panel to manage hotels and bookings
 -> Secure authentication system
 -> Responsive UI for better user experience
